# vue_todo_app

## Description

```
A simple todo_app that you can add todos, delete todos using jsonplaceholder to simulate communication with a server
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

